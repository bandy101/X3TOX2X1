# X3TOX2X1
Mac环境下使用x3图片批量生成x2、x1图片
------
###使用

```
X3TOX2X1.sh x3图片文件夹路径 生成新的文件夹路径
eg:
/Users/Ethan/Desktop/X3TOX2X1.sh /Users/Ethan/Desktop/3xpng /Users/Ethan/Desktop/3xpng_new
```
